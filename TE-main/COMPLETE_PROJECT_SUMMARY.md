## 🎯 Project Overview

### **What is This?**
A **production-ready Multi-Modal Retrieval-Augmented Generation (RAG) system** that can:
- Process **text documents** (PDF, TXT, MD, CSV)
- Analyze **images** (JPG, PNG, GIF, BMP, WEBP)
- Transcribe **audio files** (MP3, WAV, M4A, OGG, FLAC, AAC)
- Answer questions using **semantic search** and **AI generation**
